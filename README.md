# Rust学習Vault 🦀

組込みエンジニアのためのRust学習教材

## 📖 このVaultについて

このVaultは、C言語と組込み開発の実務経験を持つエンジニアが、Rustを効率的に学ぶために設計された学習教材です。

### 対象者
- C言語の実務経験（5年程度）
- 組込み開発の経験
- メモリ管理やポインタの概念を理解している方

### 学習の進め方
1. [[00-学習進捗チェックリスト]] で進捗を管理
2. [[01-CとRustの比較]] でC言語の知識を活かす
3. [[02-実践課題集]] で手を動かして学ぶ

## 🎯 学習のポイント

### Rustの核心概念
1. **所有権（Ownership）** - メモリ安全性の基盤
2. **借用（Borrowing）** - 参照の安全な使用
3. **ライフタイム** - 参照の有効期間の保証
4. **型システム** - コンパイル時の安全性

### C言語経験者の強み
- メモリレイアウトやスタック/ヒープの理解
- ポインタ操作の経験
- システムプログラミングの知識
- パフォーマンスへの意識

## 🚀 はじめ方

1. **環境構築**
   ```bash
   # Rustupのインストール
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

   # インストール確認
   rustc --version
   cargo --version
   ```

2. **最初の一歩**
   - [[00-学習進捗チェックリスト]] を開いて、学習計画を確認
   - [[01-メモリ管理の比較]] から読み始めることを推奨
   - [[課題01-計算機プログラム]] で実際にコードを書いてみる
   - エラーが出たら [[デバッグガイド]] を参照

3. **学習の習慣化**
   - 週5-10時間のペースで進める
   - 理論6割、実践4割のバランス
   - わからないことはコミュニティで質問する
   - コンパイラエラーを恐れない（エラーメッセージは最良の教師）
